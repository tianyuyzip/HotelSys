# HotelSys
基于SSH框架技术实现的web应用程序

### 系统介绍
+ 主要功能
  - 系统登录，包括密码验证、用户名验证
  - 酒店分店管理，包括分店信息录入，分店信息查询，分店信息修改（图片上传）
  - 房间管理，包括所属分店信息录入，房间信息录入，房间信息查询，房间信息修改

  <p>以上查询功能均包括分页，组合条件查询功能

+ 所用插件介绍
  - struts2
  - spring
  - hibernate
  
+ 特色功能
  - 主页通过iframe，效果类似于SPA程序
  - 分店信息录入中图片上传功能提供默认图片。
  
+ 目标要求
  <P>房间信息删除后，实现酒店信息中的房间信息级联删除。
